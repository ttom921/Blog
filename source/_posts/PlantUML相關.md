## PlantUML和Hexo整合
安装hexo-tag-plantuml
```bash
npm install hexo-tag-plantuml --save
```
## 語法
```
{% plantuml %}
start
:配置Java环境; 
:下载pantuml.jar;
:编写描述文件; 
:执行; 
stop
{% endplantuml %}
```
