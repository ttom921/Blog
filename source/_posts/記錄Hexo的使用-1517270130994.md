第一次從github上下載專案後，要先安裝hexo
``` bash
$npm install hexo -g
```
在local端測試，執行以下

``` bash
$hexo server
```
記錄完後要上傳和產生
``` bash
$hexo g
$hexo d
```
Hexo 常用簡寫命令
``` bash
$hexo n #生成文章，或者source\_posts手動編輯
$hexo n #本地發佈預覽
$hexo g #生成public靜態文件
$hexo d #部署之前生成的文件
```
